## Что за проект

Продающий одностраничный лендинг для продажи франшизы теннисной школы «Первая Школа Тенниса» (ПШТ).
Существующий сайт бренда: `bolshoitennis.ru`.

**Три целевые аудитории:** предприниматели и инвесторы (хотят купить франшизу), родители (ищут секцию для ребёнка).
Инвестор / партнёр может быть из любого города.

**Ключевое позиционирование:** «Вы заходите в поток, а не начинаете с нуля» — CRM, заявки, методика, процессы уже работают.

**Документация:**
- [PROJECT.md](PROJECT.md) — паспорт проекта, аудитории, цифры, конкуренты
- [research.md](research.md) + [research2.md](research2.md) — исследования рынка
- [PLAN.md](PLAN.md) — чек-лист задач с текущим статусом

## Правила работы с кодом

### 1. Один HTML-файл — без исключений

Весь код живёт в `franshiza-landing.html`. Никаких отдельных `.css` или `.js` файлов.

- ✅ HTML + `<style>` + `<script>` — всё внутри одного файла
- ✅ Google Fonts через `<link>` — допустимо
- ✅ SVG inline, base64 для логотипа
- ❌ Bootstrap, Tailwind, jQuery, React — не подключать
- ❌ npm, node_modules — не использовать
- ❌ Внешние JS-библиотеки без явного согласования

### 2. Перед созданием нового файла — спросить

Если задача требует нового файла — описать зачем и получить подтверждение.

### 3. Mobile-first адаптив

- Минимальная ширина: 320px
- Тач-таргеты кнопок: не менее 44px
- Нет горизонтального скролла на мобильном
- Проверять: формы, навигацию, таблицы

### 4. Обновлять PLAN.md после каждой задачи

Выполнил пункт → сразу ставь `[x]`. Не накапливай изменения.

Белый фон, синий как основной цвет, оранжевый для CTA.

> Брендбук: `Брендбук Первая школа тенниса.pdf` — страницы 1–7.

### Шрифты

- **Заголовки:** Oswald 600–700 (веб-замена Myriad Pro Cond Bold из брендбука)
- **Текст:** Inter 400–500
- Оба подключены через Google Fonts: `?family=Oswald:wght@400;500;600;700&family=Inter:wght@400;500;600`

### Визуальные принципы

- Чистый, профессиональный вид: без glow, без glassmorphism
- Карточки: белый фон, border `1px solid var(--border)`, hover — синяя рамка + тень
- Скругления: `--radius: 12px`, `--radius-sm: 8px`
- Переходы: `transition: 0.25s ease` везде
- Тени: `box-shadow: 0 4px 24px rgba(5,95,150,0.10)`

## Стиль текстов

- **Простой и прямой** — как объясняешь другу, не как пишешь в отчёт
- **Активный залог:** «мы помогаем», не «помощь осуществляется»
- **Конкретика:** цифры, сроки, примеры — не «качественный сервис»
- **Одна мысль = одно предложение**
- **Без канцелярита:** никаких «в рамках», «является», «обеспечивает», «осуществляет»

Плохо: *«В рамках сотрудничества обеспечивается предоставление маркетинговой поддержки.»*
Хорошо: *«Мы настраиваем рекламу под твой город и даём шаблоны для соцсетей.»*

## Критические проблемы (нельзя игнорировать)

1. **Политика конфиденциальности** — в форме ссылка на `#`. При сборе персданных это нарушение 152-ФЗ.
2. **Контент не обновлён.** Лендинг содержит плейсхолдеры вместо реальных данных. Реальные цифры — в docx-файлах и PROJECT.md.
3. **Нет команды на лендинге.** Инвесторы и франчайзи покупают у людей. Нужна секция с основателем.
4. **Нет WhatsApp/Telegram-кнопки.** 60–70% лидов в РФ приходят через мессенджеры.
5. **Форма не подключена к CRM.** `submitForm()` только логирует в консоль. Без webhook лиды теряются.
